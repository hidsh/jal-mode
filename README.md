# jal-mode

![Screenshot](ss.png)

**WIP**

This major mode is for editing [JAL (Just Another Language)](http://justanotherlanguage.org/).

JAL is a programming language to program Microchip PIC controllers.

This major mode will support jalv2 compiler.
